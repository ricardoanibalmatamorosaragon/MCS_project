https://pypi.org/project/memory-profiler/


*****Installation******
Install via pip:
$ pip install -U memory_profiler


*******Usage*****
from memory_profiler import profile

******Code****
@profile(precision=4)
def my_func():
    a = [1] * (10 ** 6)
    b = [2] * (2 * 10 ** 7)
    del b
    return a


******Terminal*****
mprof run <executable>
mprof plot
